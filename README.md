# Machine Learning From Scratch

This repository contains my implementations of machine learning algorithms from scratch using Python and NumPy while studying Stanford CS229.

## Goals

- Understand the mathematics behind ML algorithms
- Implement every algorithm without using ML libraries
- Build intuition through experimentation
- Document the learning journey
